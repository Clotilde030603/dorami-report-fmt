# 벤치마크 가이드

이 문서는 도라미체와 AI식 표현 완화 규칙의 효과를 비교 평가하는 개발용 기준이다.

일반 사용자가 실행해야 하는 기능이 아니며, 스킬 품질을 점검하거나 규칙을 조정할 때 사용한다.

## 비교 파이프라인

1. `original`
2. `dorami-only`
3. `ai-cleanup-only`
4. `ai-cleanup → dorami`
5. `dorami → ai-cleanup`
6. `integrated-full`

## 평가 기준

| 항목 | 점수 |
| --- | ---: |
| 의미 보존 | 30점 |
| 보고서체 적합도 | 25점 |
| 도라미체 유사도 | 20점 |
| AI식 표현 감소 | 15점 |
| 과잉 수정 없음 | 10점 |

총점은 100점으로 계산한다.

## 판단 기준

- `integrated-full`이 가장 높으면 현재 구조를 유지한다.
- `ai-cleanup → dorami`가 가장 높으면 내부 처리 순서를 해당 방향으로 조정한다.
- `dorami-only`가 가장 높으면 AI식 표현 완화 규칙이 과한 것이다.
- `ai-cleanup-only`가 가장 높으면 도라미체 규칙이 부족하거나 부자연스러운 것이다.
- `dorami → ai-cleanup`이 낮으면 후처리 구조를 금지한다.

## 검토 메모

평가할 때는 점수만 보지 않고 다음을 함께 확인한다.

- 원문의 핵심 의미가 보존되었는가
- 근거 없는 내용이 추가되지 않았는가
- 문장 표현이 과하게 바뀌지 않았는가
- 보고서 문체가 자연스럽고 객관적인가
- 반복 표현과 상투 표현이 줄었는가
