# 도라미 보고서체 Do-Not

## 표현 관련 금지·주의 항목

- “현대 사회에서 매우 중요한 문제로 대두되고 있다” 같은 상투적인 도입부를 쓰지 않는다.
- “매우”, “획기적”, “필수불가결”, “시사하는 바가 크다”를 남발하지 않는다.
- 근거 없는 확정 표현을 쓰지 않는다.
- 구어체, 감탄형, 감상문처럼 보이는 표현을 피한다.
- 홍보성 표현이나 광고 문구처럼 보이는 수식어를 사용하지 않는다.
- 같은 문장 구조와 같은 종결어미를 반복하지 않는다.
- 예시 문장이나 참고 문장을 그대로 복사하지 않는다.

## 정보 보호 관련 금지 항목

- 개인정보가 들어간 실제 보고서 예시를 공개 저장소에 포함하지 않는다.
- `.hwp`, `.hwpx`, 변환된 Markdown, 익명화 예시 파일을 공개 저장소에 포함하지 않는다.
- `private/`, `local-references/`, `converted-examples/`, `redacted-examples/`, `raw/`, `tmp/` 안의 파일을 커밋하지 않는다.

## 목적 설명 관련 금지 항목

- 이 스킬을 문서 판별 시스템 회피 목적으로 설명하지 않는다.
- 목적은 “AI식 상투 표현 완화”, “부자연스러운 문장 정리”, “보고서 문체로 다듬기”로 설명한다.

## 공개 배포 안전 관련 금지 항목

- AI식 도입부를 반복적으로 사용하지 않는다.
- 근거 없는 과장이나 확정 표현을 사용하지 않는다.
- 같은 연결 표현, 같은 종결어미, 같은 문장 구조를 반복하지 않는다.
- 보고서와 맞지 않는 구어체나 감상문처럼 보이는 표현을 사용하지 않는다.
- 홍보성 표현으로 정책·기술 효과를 과대하게 설명하지 않는다.
- 실제 보고서 예시 문장이나 개인 자료를 공개 파일에 복사하지 않는다.
- 로컬 자료나 분석 결과를 공개 폴더로 옮기지 않는다.
- 문서 판별 시스템 회피를 목적으로 하는 표현을 사용하지 않는다.
